## Place three Stocks

Press `4` (or click the Stock button in the toolbar), then click on the canvas three times, arranging them left to right:

1. **Susceptible** — initial value `990`
2. **Infectious** — initial value `10`
3. **Recovered** — initial value `0`

Total population is 1000. We start with 10 infected people and everyone else susceptible.

Double-click each stock to name it and set the initial value in the Properties panel on the right.
