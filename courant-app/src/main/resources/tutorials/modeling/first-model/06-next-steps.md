## What you learned

- **Stock** — accumulates a quantity (Coffee Temperature)
- **Flow** — changes a stock each time step (Cooling)
- **Variable** — computes a derived value (Discrepancy)
- **Parameter** — a variable with a fixed value (Room Temperature, Cooling Rate)
- **Negative feedback** — the system self-corrects toward equilibrium

These five ideas can model surprisingly complex systems — from disease epidemics to supply chains to climate dynamics.
