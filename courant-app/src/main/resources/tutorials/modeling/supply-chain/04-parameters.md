## Place four Parameters

Press `6` to switch to the Variable tool. Create:

1. `Customer_Demand` = `20` (units per day)
2. `Target_Inventory` = `200` (units — same as initial inventory)
3. `Adjustment_Time` = `4` (days to correct an inventory gap)
4. `Delivery_Delay` = `5` (days from order to receipt)

The system starts in equilibrium: inventory equals target, demand is steady, and orders equal demand.

## Validate

Press `Ctrl+B` to check for errors before running.
