## Set Link Polarity

Every causal link has a **polarity** that describes the direction of influence:

- **+** (positive) — when the cause increases, the effect increases (all else being equal)
- **−** (negative) — when the cause increases, the effect decreases
- **?** (unknown) — direction not yet determined

Press `1` to switch to the Select tool. Right-click each link and choose **Set Polarity**:

1. Cars on Road → Congestion: **+**  (more cars → more congestion)
2. Congestion → Travel Time: **+**  (more congestion → longer trips)
3. Travel Time → Public Transit Use: **+**  (longer trips → more people switch)
4. Public Transit Use → Cars on Road: **−**  (more transit → fewer cars)
5. Congestion → Road Construction: **+**  (more congestion → more road building)
6. Road Capacity → Congestion: **−**  (more capacity → less congestion)

**Important:** Polarity describes the *direction* of the relationship, not its sign in an equation. "More A leads to less B" means the polarity is negative.
