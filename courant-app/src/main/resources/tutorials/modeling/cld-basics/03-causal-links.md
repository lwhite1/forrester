## Draw Causal Links

Click the **Causal Link** button in the toolbar to switch to the Causal Link tool.

Drawing a link is a two-click operation: click the **cause** variable, then click the **effect** variable.

Draw these six links:

1. `Cars on Road` → `Congestion`
2. `Congestion` → `Travel Time`
3. `Travel Time` → `Public Transit Use`
4. `Public Transit Use` → `Cars on Road`
5. `Congestion` → `Road Construction`
6. `Road Capacity` → `Congestion`

**Tip:** If you make a mistake, press `1` to switch back to the Select tool, right-click the link, and choose Delete.
