Shrewd

An Open-Source Environment for System Dynamics Simulation

The Shrewd Project is an implementation of System Dynamics (SD) principles designed for structural transparency and numerical rigor. Unlike general-purpose data science tools, Shrewd is specialized for the modeling of stocks, flows, and the feedback loops that define non-linear behavior over time.

For the researcher, Shrewd provides a bridge between visual sketching and formal mathematical specification. For the student, it offers an unencumbered environment to explore the foundations of cybernetics and feedback theory without licensing constraints.

Numerical Integration and Stability

Simulation in Shrewd is handled by a high-performance solver supporting Euler and Runge-Kutta (RK4) integration methods. As model complexity increases—particularly with subscripts and multi-dimensional arrays—computational efficiency becomes paramount. Shrewd is optimized for the Java Virtual Machine, allowing for sub-millisecond execution of large-scale models.

Causal Loop Identification: The environment automatically parses the underlying directed graph of a model to identify reinforcing and balancing feedback loops. This allows for immediate verification of the system's behavioral modes: exponential growth, goal-seeking, or oscillation.

Uncertainty and Sensitivity

Deterministic models provide a single trajectory, but they often fail to capture the sensitivity of a system to parameter variance. Shrewd integrates Monte Carlo simulation as a core primitive. By assigning probability distributions to exogenous variables, researchers can generate an envelope of outcomes, providing a more statistically robust view of policy resistance.
