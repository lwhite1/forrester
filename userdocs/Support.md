# Support

Thank you for using this Forrester. To ensure the sustainability of this open-source effort while managing a limited time budget, we use a community-driven support model.

------

## 🔍 Before You Ask

Before reaching out, please follow these steps to see if your question has already been addressed:

1. **Read the Documentation:** Most common setup questions and configuration details are covered in our core docs.
2. **Search GitHub Discussions:** Check the **Discussions** tab to see if another user has encountered and solved the same problem.
3. **Check Open Issues:** Search our **Issue Tracker** to see if a bug you’ve found is already being investigated.

------

## 💬 Where to Get Help

We utilize **GitHub Discussions** as our primary forum for support. Please follow these guidelines to ensure you get the right kind of help:

### GitHub Discussions (Usage Questions)

If you have a question about how to use the software, need help with a specific configuration, or want to share an idea, please start a new thread in **Discussions**.

- **Community Powered:** This project relies on users helping users. While maintainers participate when possible, "community support only" is the standard expectation.
- **Searchable:** Keeping usage questions in Discussions helps build a knowledge base for future users.

### GitHub Issues (Bug Reports & Feature Requests)

The **Issues** tab is strictly for tracking actionable work.

- **Bug Reports:** If the software is crashing or behaving incorrectly, please file an issue with a reproducible example.
- **Issues are NOT for Support:** If you post a "How do I..." question in the Issues tab, it will likely be closed and redirected to Discussions.

------

## 🛠 Support Expectations

As this is a volunteer-led open-source project, please keep the following in mind:

- **No Guaranteed Response Time:** Support is provided on a "best effort" basis by the community.
- **Be Descriptive:** When asking for help, provide your environment details, version number, and what you have already tried.
- **Contribute Back:** If you find the answer to your own question, please update your Discussion thread so others can benefit!
